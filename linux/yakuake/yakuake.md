# yakuake

[Website](https://apps.kde.org/en/yakuake)

- `ctl + d` close active terminal (default `ctl + shift + r`)
- `ctl + shift + up | down` switch to next terminal
- `ctl + (` split to right
- `ctl + )` split to bottom
- `Super + a` open yakuake (default `F12`)
